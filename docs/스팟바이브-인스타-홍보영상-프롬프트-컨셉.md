# 스팟바이브 — 인스타 홍보용 영상 프롬프트·컨셉 모음

**용도:** 인스타그램 릴스·숏츠·스토리용 **홍보 영상**을 만들 때, 제작자(또는 협업 크리에이터·팔로워)에게 **그대로 전달하거나 살짝만 수정**해 쓸 수 있는 **기초 자료**입니다.  
**한국어 애니메이션·모션 그래픽·웹툰풍** 등 여러 톤을 섞었고, 영상 생성 AI(예: 영상 모델·이미지→영상 도구)마다 입력 방식이 다르므로 **「영상 프롬프트」**와 **「비주얼 참고용」**을 구분해 두었습니다.

---

## 1. 브랜드·메시지 고정값 (모든 영상 공통)

| 항목 | 내용 |
|------|------|
| 앱 이름 | **스팟바이브** (SpotVibe) |
| 한 줄 | 지금 이 근처 **핫스팟·이벤트·인파 바이브**를 지도로 본다 |
| 키 비주얼 | 다크 모드 지도, 노랑·시안·핑 포인트, 점(군집), 이벤트 핀, **「여기 어때?」** AI 말풍선, 심장박동 원, AI 추천 카드 |
| 핵심 기능 한 줄 | AI가 내 성향에 맞는 장소를 화살표로 안내하고 30초마다 다음 추천을 보여준다 |
| CTA 예시 | 프로필 링크 / 바이오 링크, 「AI 추천 눌러봐」, 「지도에서 바로 볼래?」 |

**금지·주의 (홍보 시):** 과장된 실시간 수치, 허위 이벤트, 타사 앱 UI 그대로 복제, 저작권 있는 지도·캐릭터 직접 모방.

## 3. 컨셉별 시나리오 + 프롬프트

각 컨셉마다 **로그라인 → 씬 요약 → 영상용 프롬프트(한글)** 순입니다.

### 컨셉 A — 「오늘 어디 갈지」15초 오프닝 (숏애니)

**로그라인:** 친구 둘이 카톡하다 스팟바이브 켜고 지도가 살아난다.

**씬:** ① 방 안 소파 + 폰 ② 지도 화면 줌인 ③ 점들이 반짝 ④ 「여기 어때?」 말풍선 ⑤ 둘이 밖으로 뛰어나감(실루엣).

**영상 프롬프트 (한글):**

```
컨셉 A. 2D 한국 애니, 15초, 9:16.
방 안에서 스마트폰을 보는 20대 두 명(남녀 혼합 가능), 카톡 말풍선은 한글만.
화면이 지도 앱으로 전환되며 어두운 배경에 반짝이는 점들과 이벤트 핀이 나타남.
화면 중앙에 작은 말풍선 "여기 어때?" 가 떴다가 화살표가 한 곳을 가리킴.
마지막 2초: 두 사람이 문을 열고 밤거리로 나가는 뒷모습, 상단에 작게 "스팟바이브" 로고 느낌의 텍스트.
```

### 컨셉 B — 「지도가 말 걸어요」귀여운 UI 요정

**로그라인:** 지도 속 작은 요정(또는 빛 덩어리)이 스팟을 소개한다.

**씬:** ① 지도 전체 ② 캐릭터 등장 ③ 핀 하나씩 톡톡 ④ MBTI/시간대 UI가 살짝 스쳐 지나감 ⑤ CTA 자막.

**영상 프롬프트 (한글):**

```
컨셉 B. 치비 캐릭터 또는 빛나는 구체 요정, 한국어 애니풍, 12초, 9:16.
가상의 다크모드 지도 위를 날아다니며 이벤트 핀을 손가락으로 툭툭 친다.
말 없이 행동만, 자막으로만 한국어: "지금 이 근처", "핫한 거 여기", "스팟바이브".
배경은 홍대·연남 느낌의 단순화된 야경 실루엣만(특정 상호명 없음).
```

### 컨셉 C — 「MBTI가 지도를 고른다면」패러디

**로그라인:** ENFP vs ISTJ가 같은 지도를 봐도 다른 핀을 고른다.

**씬:** 분할 화면 → 각자 탭 → 한곳에서 만남(웃음).

**영상 프롬프트 (한글):**

```
컨셉 C. 코믹 한국 웹툰 스타일, 18초, 9:16, 화면 분할.
왼쪽 ENFP 스타일 캐릭터(활발) 오른쪽 ISTJ 스타일 캐릭터(차분),
둘 다 같은 스타일의 가짜 지도 앱 UI. 왼쪽은 붉은 점 많은 쪽, 오른쪽은 조용한 카페 핀.
마지막에 중앙에서 만나 쌍둥이처럼 같은 핀을 가리키며 놀람. 자막: "스팟바이브는 취향대로".
```

### 컨셉 N — 「심장 뛰는 지도」오버레이 시각화 *(신규)*

**로그라인:** 지도가 살아있는 것처럼 원들이 심장박동처럼 맥박 친다.

**씬:** ① 어두운 지도 클로즈업 ② 반투명 원이 살짝 커졌다 줄어드는 루프 ③ 군집 점들이 흐르듯 이동 ④ 자막.

**영상 프롬프트 (한글):**

```
컨셉 N. 순수 모션 그래픽, 루프 가능, 10초, 9:16.
어두운 배경 위 반투명 시안·골드·핑크 원들이 심장박동처럼 살짝 커졌다 작아지는 루프 애니.
중심에서 작은 점들이 물처럼 천천히 흐름.
BGM은 낮고 리드미컬한 베이스 심박 사운드.
자막 없이 마지막 1초에만 "스팟바이브" 로고 페이드인.
```

---

### 컨셉 O — 「여기 어때? 말풍선」깜짝 등장 *(신규)*

**로그라인:** 지도를 보다가 갑자기 AI 말풍선이 떴다 사라진다 — 매번 다른 장소.

**씬:** 정적인 지도 → 말풍선 팝 → 화살표 + 장소명 → 20초 후 사라짐 → 또 다른 말풍선.

**영상 프롬프트 (한글):**

```
컨셉 O. 짧은 루프 애니, 20초, 9:16, 코믹 + 신기한 톤.
어두운 지도 위에 말풍선이 "뽕" 하고 등장(효과음 O).
말풍선 한글 텍스트: "여기 어때?" 또는 "여기는?" (2회 교차).
점선 경로와 회전 화살표 아이콘이 한 지점을 가리킴.
말풍선 아래엔 장소명 + 한 줄 설명 자막.
20초 후 말풍선이 사라지고 화면 하단 자막: "AI가 가끔 귀띔해줘요, 스팟바이브".
```

---

### 컨셉 P — 「운영자에게 직접 피드백」커뮤니티 신뢰 *(신규)*

**로그라인:** 사용자가 개선 의견을 보내고, 그게 실제로 반영된다는 신뢰 메시지.

**씬:** 마이 탭 → 문의 버튼 탭 → 메일 초안 자동 완성 → 보내기 → 체크표시 + 따뜻한 자막.

**영상 프롬프트 (한글):**

```
컨셉 P. 가벼운 2D 한국 애니, 15초, 9:16, 따뜻한 톤.
가상 앱 마이 탭의 "개선 제안" 버튼을 캐릭터가 탭.
하단 다이얼로그가 올라오며 텍스트 자동 입력 효과.
"메일 앱으로 보내기" 탭 후 종이비행기 애니메이션.
자막: "여러분 의견이 스팟바이브를 만들어요" + 하트.
```
