## 1. 실시간 점 이동 — 1분 단위 군중 흐름 애니메이션

### 아이디어
지도 위 연령대 점(20대·30대·40대)이 1분마다 약간씩 이동해, **사람들이 실시간으로 움직이는 흐름**처럼 보이게 한다.

### 구현 방향
- **프론트(현재 가능)**: PRNG 시드(seed)를 1분마다 갱신 → 점 좌표가 자연스럽게 drift  
  Framer Motion `animate`로 각 점이 새 좌표로 부드럽게 이동.
- **백엔드 연동 시**: 서버에서 1분 집계 히트맵 데이터를 WebSocket/SSE로 푸시  
  클라이언트는 수신 즉시 `flyTo` + 점 업데이트.

### 판단: 1분 단위가 적절한가?
| 주기 | 장점 | 단점 |
|------|------|------|
| 실시간(5~10초) | 생동감 극대 | 배터리·서버 부하 ↑, 사용자 현기증 |
| **1분** | 변화 체감 + 부하 균형 ✅ | 아주 빠른 이벤트 반영 늦음 |
| 5분 | 서버 최소 부하 | 변화 체감 어려움 |

→ **1분 단위가 최적**. 큰 이벤트 발생 시 "급증 알림"으로 보완 가능.

## 2. MBTI·성향 기반 집합 시각화

### 아이디어
사용자가 MBTI(또는 취향 태그)를 설정하면, 지도 위에  
- **나와 같은 성향이 많은 구역** = 원(집합) 으로 표시  
- **교집합** = 두 성향이 겹치는 구역 (색이 혼합된 반투명 오버레이)  
- **합집합** = 두 그룹 중 어느 하나라도 있는 전체 범위

### 수학적 구조
```
A = MBTI E(외향)가 밀집한 구역
B = 20대 여성이 밀집한 구역

A ∩ B = 내가 설정한 두 조건을 모두 만족하는 구역  → 가장 진한 색 강조
A ∪ B = 어느 조건이든 해당되는 구역            → 연한 색 오버레이
A \ B = E성향만 많고 20대 여성은 적은 구역       → 별도 색
```

### 구현 방향
- **마이페이지**에 MBTI 16가지 + 취향 태그(클럽/공연/야외/카페 등) 선택
- 선택된 조건을 조합 → **교집합 원**, 합집합 원을 Leaflet `Circle` + 투명도로 표현
- 조건이 많을수록 원이 작아지고 색이 진해짐 (정밀 타게팅 느낌)
- 백엔드: 실제 사용자 데이터 기반 클러스터링 필요 (현재는 더미 PRNG로 시뮬레이션)

### 특허 가능성
- **"다중 사용자 성향 속성을 교집합 집합 연산으로 지리 군집 오버레이를 생성하는 방법"** 은 구체 알고리즘 + 기술 효과가 명확하면 출원 검토 가치 있음 → `스팟바이브-특허-및-IP-참고.md` 2절 참고.

## 3. 실시간 현장 사진 업로드 + 자동 모자이크

### 아이디어
사용자가 현장에서 사진을 찍어 지도 위 해당 위치에 올리면,  
**얼굴·간판·차량 번호판 등 개인식별 요소가 자동 블러·모자이크** 처리된 뒤 다른 사용자에게 노출된다.

### 기술 스택 (구현 시)
```
[클라이언트]
1. 카메라 캡처 (MediaStream API)
2. EXIF GPS 메타데이터 추출 → 지도 핀 위치 자동 세팅
3. 프리뷰 UI (얼굴 감지 결과 바운딩박스 미리보기)

[서버 파이프라인]
4. 이미지 업로드 → S3/CDN
5. 얼굴 감지: AWS Rekognition / Google Vision API / ONNX face_detection 모델
6. 번호판 감지: YOLOv8 finetuned 모델
7. 감지된 영역 → 서버사이드 Gaussian Blur / Pixelate 처리
8. 처리된 이미지 → CDN 배포
9. WebSocket / SSE → 주변 사용자에게 "새 현장 사진" 알림

[법·정책]
- 정보통신망법, 개인정보보호법 상 "자동 비식별화" 의무 처리 필수
- 모자이크 처리됐더라도 게시 전 AI 검수 + 신고 기능 병행 권장
- 이용약관에 "위치 정보 수집 동의" 항목 필요
```

### 프론트 프로토타입 (현재 가능)
- 지도 위 **카메라 핀 버튼** 추가
- 더미 "현장 사진 카드" (블러 처리 UI 시뮬레이션)를 지도에 오버레이
- 탭하면 사진 상세 바텀시트 (업로드 시각, 거리, 모자이크 안내 텍스트)

### 특허 가능성
- **"위치 메타데이터 기반 실시간 현장 사진의 자동 비식별화 및 지도 레이어 통합 시스템"** 은 파이프라인 전체를 구체 도면+실시예로 기술하면 **출원 검토 가치 높음**.
