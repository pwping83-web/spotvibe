# 스팟바이브 — 인스타 홍보용 영상 프롬프트·컨셉 모음

**용도:** 인스타그램 릴스·숏츠·스토리용 **홍보 영상**을 만들 때, 제작자(또는 협업 크리에이터·팔로워)에게 **그대로 전달하거나 살짝만 수정**해 쓸 수 있는 **기초 자료**입니다.  
**한국어 애니메이션·모션 그래픽·웹툰풍** 등 여러 톤을 섞었고, 영상 생성 AI(예: 영상 모델·이미지→영상 도구)마다 입력 방식이 다르므로 **「영상 프롬프트」**와 **「비주얼 참고용」**을 구분해 두었습니다.

### 1-1. 모든 영상에 **반드시** 강제되는 4대 조건

> 영상 AI에 프롬프트를 넣을 때 컨셉별 문장과 함께 아래 4줄을 **항상** 첫머리에 포함합니다. 이 조건은 시리즈 일관성을 위한 절대 규칙입니다.

1. **한국 배경** — 한국 야경(서울 골목, 한강변 실루엣, 홍대·연남·강남 분위기). 일본 도쿄·중국 상하이·서양 도시 금지. 간판·표지판은 **무문자 실루엣**으로만 처리(한글 표기는 후편집).
2. **한국인 인물** — 20대 한국인 외모(밝은 피부, 검은~다크브라운 머리, 자연스러운 동양 이목구비). **일본 애니메이션 특유의 큰 눈·과장 표현 금지**, K-웹툰 톤 유지.
3. **음성·대사·BGM 없음 (무음 영상)** — 캐릭터 입은 움직이지 않거나 닫혀 있고, 더빙·내레이션·환경음·음악 모두 제외. 텍스트 정보는 자막·말풍선·UI 모션으로만 전달. (BGM·효과음은 **편집자가 후작업으로** 저작권 프리 음원을 입힘.)
4. **동일 인물 유지** — 모든 컨셉(A~P)에서 등장 인물은 §3 「캐릭터 바이블」에 정의된 **주인공 2인(지우·민호) 고정**. 같은 의상·머리·체격을 유지해야 시리즈 인지도가 쌓임.

**금지·주의 (홍보 시):** 과장된 실시간 수치, 허위 이벤트, 타사 앱 UI 그대로 복제(카카오맵·네이버지도·구글맵 UI 직카피 금지), 저작권 있는 지도·캐릭터 직접 모방, 실존 상호명·로고 노출.

## 2. 공통 「스타일」 프롬프트 블록 (복붙 후 컨셉별 문장만 붙이기)

아래 블록을 **영상 AI의 스타일 프롬프트** 앞에 그대로 붙여 쓰면 §1-1의 4대 조건이 자동 반영됩니다.

- 등장 인물은 20대 한국인 2인(주인공 지우·민호, 캐릭터 바이블 §3 참조), 동일 외모·의상 유지.
- 한국 K-웹툰·셀 애니메이션 스타일, 깔끔한 선화, 부드러운 셀 쉐이딩.
- 컬러: 다크 배경 + 네온 포인트(시안 #00E5FF, 골드 #FFD24A, 핑크 #FF5C8A).

- 한글 자막은 영상 AI에 직접 넣지 말고 **후편집 추가**(AI가 한글을 깨뜨리는 경우가 많음).
[금지]
- 일본 애니풍 큰 눈·코스프레, 과도한 성형 표현, 폭력·혐오·선정적 묘사,
- 실존 브랜드/연예인 얼굴, 실제 카카오맵·네이버지도 UI, 깜빡임·스트로보,

```

## 3. 캐릭터 바이블 (모든 컨셉에서 동일 인물 유지) ⭐신규

> 영상 AI는 컷마다 인물 외모가 흔들리기 쉽습니다. **첫 컷을 레퍼런스 이미지로 저장**한 뒤 이후 컷에 시드/이미지 가이드로 재투입하세요. 캐릭터 묘사를 **매 프롬프트에 동일한 문장으로 복붙**하는 것이 핵심입니다.

### 3-1. 주인공 「지우」 (여, 24세)

```
한국인 여성, 24세. 단발~어깨길이 검은 머리(자연스러운 웨이브),
밝은 피부, 작은 코, 또렷한 눈매(일본 애니풍 과장 없음).
의상: 오버사이즈 베이지 후디 + 데님 와이드 팬츠 + 흰 운동화,
크로스백, 무선 이어폰. 키 165cm 비율.
표정: 호기심·미소 위주, 폰을 자주 들여다봄.
```

### 3-2. 주인공 「민호」 (남, 25세)

```
한국인 남성, 25세. 짧은 검은 머리(투블럭 살짝),
보통 피부톤, 둥근 안경, 부드러운 인상.
의상: 다크 그레이 맨투맨 + 검은 슬랙스 + 검은 스니커즈,
손목에 스마트워치. 키 178cm 비율.
표정: 차분, 가끔 놀란 듯 눈썹 올림.
```

### 3-3. 보조: UI 요정(컨셉 B·N·O 한정)

```
손바닥 크기의 시안색 빛 구체, 작은 날개 또는 꼬리 잔상,
표정 없음, 캐릭터화 시 치비형(머리 큰 2등신) 별도 사용.
주인공 2인과 같은 화면에 등장 가능, 인간 캐릭터 외모는 변경 금지.
```

### 3-4. 인물 일관성 유지 체크리스트

- [ ] 컷마다 **같은 의상** (지우=베이지 후디, 민호=다크 맨투맨) 유지
- [ ] 첫 생성 컷을 **레퍼런스 이미지**로 저장 후 다음 컷 입력에 첨부
- [ ] 영상 AI가 시드 고정 옵션을 지원하면 **동일 시드** 사용
- [ ] 얼굴 클로즈업은 최소화(AI가 가장 잘 흔들리는 구간) — 미디엄/와이드 샷 권장
- [ ] 인물이 1명만 나와도 좋음. 단, 등장 인물은 반드시 지우 또는 민호 중 1명

## 4. 컨셉별 시나리오 + 프롬프트

> **사용 방법**: 각 컨셉 프롬프트 앞에 **§2 [고정 조건] 블록 + §3 캐릭터 바이블 해당 인물 묘사**를 반드시 함께 입력하세요. 그래야 4대 강제 조건(한국 배경·한국인·무음·동일 인물)이 유지됩니다.

각 컨셉마다 **로그라인 → 씬 요약 → 영상용 프롬프트(한글)** 순입니다.

---

### 컨셉 A — 「오늘 어디 갈지」15초 오프닝 (숏애니)

**로그라인:** 지우와 민호가 메시지를 주고받다 스팟바이브를 켜고 지도가 살아난다.

**씬:** ① 한국식 원룸 거실 + 폰 ② 지도 화면 줌인 ③ 점들이 반짝 ④ 「여기 어때?」 말풍선 ⑤ 둘이 밤거리로 뛰어나감(실루엣).

**영상 프롬프트 (한글):**

```
컨셉 A. K-웹툰 2D 애니, 15초, 9:16, 무음.
[등장] 주인공 2인 — 지우(베이지 후디 여성), 민호(다크 맨투맨 안경 남성). §3 캐릭터 바이블 동일 외모 유지.
[배경] 한국 원룸 거실, 창밖에 서울 야경 실루엣(가로등·아파트 윤곽, 간판은 무문자).
[액션] 지우가 소파에서 폰을 보고, 민호가 옆에서 같은 화면을 들여다봄.
대사·립싱크 없음. 메시지 말풍선만 떠오름(텍스트는 후편집).
화면이 지도 앱으로 전환되며 다크 배경에 시안·골드 점이 반짝이고 이벤트 핀이 솟아남.
중앙에 작은 말풍선 "여기 어때?" → 화살표가 한 지점을 가리킴.
마지막 2초: 지우·민호가 문을 열고 밤거리로 나가는 뒷모습, 상단에 "스팟바이브" 자리(로고는 후편집).
```

### 컨셉 B — 「지도가 말 걸어요」귀여운 UI 요정

**로그라인:** 지도 속 작은 시안색 빛 요정이 스팟을 소개한다(인간 주인공은 손만 등장).

**씬:** ① 지도 전체 ② 요정 등장 ③ 핀 하나씩 톡톡 ④ MBTI/시간대 UI가 살짝 스쳐 지나감 ⑤ CTA 자막.

**영상 프롬프트 (한글):**

```
컨셉 B. 12초, 9:16, 무음.
[등장] §3-3 UI 요정(시안색 빛 구체) + 지우의 손 일부(베이지 후디 소매).
[배경] 가상 다크모드 지도. 한국 홍대·연남풍 야경 실루엣을 지도 가장자리에 단순 배치(상호명·간판 텍스트 없음).
[액션] 요정이 지도 위를 날아다니며 이벤트 핀을 톡톡 친다. 대사·음성 없음.
자막은 후편집으로 추가 예정 — 영상 AI에는 자막을 직접 그리지 말고 빈 영역만 확보:
  "지금 이 근처" / "핫한 거 여기" / "스팟바이브" 가 들어갈 자리(화면 하단 1/4).
```

### 컨셉 C — 「MBTI가 지도를 고른다면」패러디

**로그라인:** ENFP 지우 vs ISTJ 민호가 같은 지도를 봐도 다른 핀을 고른다.

**씬:** 분할 화면 → 각자 탭 → 한곳에서 만남.

**영상 프롬프트 (한글):**

```
컨셉 C. 코믹 K-웹툰 스타일, 18초, 9:16, 무음, 화면 좌우 분할.
[등장] 왼쪽 = 지우(ENFP 톤, 활발한 표정·점프 모션), 오른쪽 = 민호(ISTJ 톤, 차분·끄덕임).
        §3 캐릭터 바이블 동일 외모·의상 엄수.
[배경] 양쪽 모두 한국 야경 톤의 가짜 지도 앱 UI(다크 배경 + 시안·골드 핀).
[액션] 왼쪽은 붉은 군집 점이 많은 핫스팟, 오른쪽은 조용한 카페 핀.
대사·음성 없음. 표정과 손짓으로만 감정 전달.
마지막에 분할이 사라지며 둘이 중앙에서 만나 같은 핀을 동시에 가리키며 놀람(입은 닫혀 있음).
자막 자리: "스팟바이브는 취향대로"(후편집 입력).
```

### 컨셉 D — 「1분 전 아직 몰랐던 스팟」미스터리 톤 (가벼운 버전)

**로그라인:** 알림이 울리기 전엔 몰랐던 근처 이벤트를 지우가 발견한다.

**씬:** ① 시계 ② 알림 배너(가짜 UI) ③ 지도 줌 ④ 행사 썸네일 스타일 일러스트 ⑤ 웃으며 출발.

**영상 프롬프트 (한글):**

```
컨셉 D. 가벼운 미스터리 코미디 K-웹툰, 15초, 9:16, 무음.
[등장] 지우(베이지 후디) 단독. §3 캐릭터 바이블 외모 유지.
[배경] 한국 원룸의 어두운 방, 창밖에 서울 아파트 야경 실루엣.
[액션] 폰 알림 배너 등장 — 텍스트 영역만 비워두고 후편집으로 한글 채움(브랜드명 없음).
지도가 확대되며 "지금 막 올라온 근처 이벤트" 느낌의 빛나는 시안 핀.
지우의 표정이 반전되어 베이지 후디 위에 외투를 입고 미소(입 닫힘).
음성·BGM·효과음 없음. 톤은 무섭지 않게 밝게 마무리.
```

### 컨셉 E — 「손 안의 홍대」POV (1인칭)

**로그라인:** 시청자 = 지우의 시점. 손가락으로 지도를 밀고 당김.

**씬:** POV 손 → 스와이프 → 핀 터치 → 하단 시트 슬라이드 업(가짜 UI).

**영상 프롬프트 (한글):**

```
컨셉 E. 1인칭 POV, 모션 그래픽 위주, 14초, 9:16, 무음.
[등장] 화면에 보이는 것은 지우의 손 일부(베이지 후디 소매가 살짝 보임). 손가락 5개·정확한 비율 엄수.
[배경] 한국 야경 카페 안 분위기(테이블 가장자리·따뜻한 조명, 간판 텍스트 없음).
[액션] 손이 스마트폰 지도를 줌인·스와이프. 가상 다크 테마 지도(시안·골드 핀).
핀을 누르자 아래에서 카드 UI가 올라옴 — 카드 안 텍스트 영역은 비워두고 후편집.
대사·음성 없음, 손가락 왜곡 금지(6개 손가락·뒤틀림 방지).
마지막 자막 자리: "내 손안 핫스팟, 스팟바이브"(후편집 입력).

## 5. 제작자에게 넘길 때 체크리스트

### 5-1. 영상 AI 입력 단계

- [ ] §1-1 **4대 강제 조건**(한국 배경·한국인·무음·동일 인물) 4줄을 프롬프트 첫머리에 포함
- [ ] §2 **[고정 조건] 블록**과 §3 **캐릭터 바이블** 인물 묘사를 함께 입력
- [ ] **9:16** 고정, **12~25초** 권장
- [ ] 음성·BGM·SFX **모두 영상 AI에 입력 금지** (편집 단계에서 처리)
- [ ] 한글 자막은 영상 AI에 직접 그리지 말 것 → 텍스트 자리(빈 박스)만 확보
- [ ] 첫 컷에서 인물이 잘 나오면 **레퍼런스 이미지 저장** → 후속 컷 입력에 첨부

## 8. 문서 활용 방법

1. **§7 마스터 프롬프트** 복사 → 그 아래 컨셉별 프롬프트(§4 A~P 중 1개) 붙여넣기 → 한 덩어리로 영상 AI에 입력
   - 신규 기능 강조 시: 컨셉 M(AI 추천 버튼), N(심장박동), O(말풍선 인도), P(운영 문의) 우선 사용
2. 사용 중인 **영상 AI 도구**에 맞게 길이·fps만 조정 (인물 일관성 약한 도구는 첫 컷을 레퍼런스 이미지로 사용)
3. 모든 컷 생성 후 **인물 외모·의상 일치 여부** 검수(불일치 컷은 재생성)
4. **자막·BGM은 후편집에서 추가** (Premiere·CapCut·DaVinci 등)
5. **썸네일**은 §2 이미지용 한 줄 + 컨셉 키워드 3개 추가
6. 내부 검수 후 **브랜드 가이드·법무** 확인(특히 B2B·수치 표현)

## 9. 영상 AI 일반 약점 — 반드시 후편집으로 보정 ⭐신규

| 약점 | 빈도 | 대응 |
|---|---|---|
| 한글 자막이 깨지거나 외계어로 나옴 | **매우 높음** | 영상 AI에 한글을 그리지 말고 빈 텍스트 박스만 만들고 **후편집에서 한글 자막 입힘** |
| 컷 사이 인물 외모가 달라짐 | 높음 | 첫 컷 레퍼런스 이미지 저장 → 후속 컷에 첨부, 동일 시드 사용 |
| 손가락 6개·뒤틀린 손 | 중간 | 손 클로즈업 회피, 미디엄 샷 위주, 이상 컷은 잘라내기 |
| 얼굴이 일본 애니풍·서양 미인풍으로 빠짐 | 높음 | "K-웹툰 톤, 일본 애니 과장 금지"를 매 프롬프트에 명시 |
| 도시 배경이 도쿄·뉴욕풍으로 빠짐 | 높음 | "Seoul night, Hongdae alley, no Tokyo/Shanghai"를 영문 병기 |
| 입이 움직이며 말하는 듯 보임 | 중간 | "mouth closed, no lipsync, no voice"를 negative에 명시 |
| 음성·SFX가 무작위로 들어감 | 낮음 | 영상 AI에서 사운드를 추출 가능하면 무음 옵션 ON, 안 되면 후편집에서 음성 트랙 제거 |
