## SpotVibe 기준 요약

| 구분 | 현재 스택에서 볼 곳 |
|------|---------------------|
| 프론트 배포 | Vite 빌드 → Netlify / Vercel / S3+CloudFront 등 (팀이 쓰는 호스팅 대시보드) |
| API·DB·실시간 | **Supabase** (Postgres, Auth, Realtime, Storage) |
