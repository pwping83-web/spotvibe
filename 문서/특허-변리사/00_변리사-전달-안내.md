## 핵심 발명 요약 (변리사 전달용 한 단락)

SpotVibe는 위치 기반 군집 밀도 시각화 서비스 내에 (1) **AI 검증 현장 제보 시스템** (텍스트 LLM + EXIF 무결성 + 비전 LLM + 집단 신고 연동 다층 검증), (2) **하이퍼로컬 이웃 도움 신호(SOS) 시스템** (4유형 분류·공식 기관 선행 안내·만료 투명도 감쇠·진동/푸시 귀환 유도), (3) **일상 DAU 기반 수동 관찰자→능동 구조자 전환 BM** (군집 탐색 습관→SOS 사례 사진 사회적 증거→바이럴 루프) 을 결합한 모바일 웹 플랫폼입니다. 각 축별 청구 가능성을 검토 요청드립니다.

## 현행 구현 기준 (출원 명세 통일 주의사항)

| 항목 | 초기 기획안 | **현행 코드 기준 (우선)** |
|------|-------------|--------------------------|
| SOS 유형 수 | 8유형 (초기 기획) | **4유형**: fire / public_safety / missing / medical |
| 지도 엔진 | Kakao Maps SDK (예정) | **Leaflet + Carto** (Kakao는 로드맵) |
| SOS 반경 | 1km (기획안 일부) | **3km** (`RADIUS_KM = 3`) |
| 모더레이션 상태 | allow/block | held 포함 **3단계** (spot) / 2단계 (sos_note) |

> 명세·도면은 **현행 코드 기준**으로 작성해야 실시예 정합성이 유지됩니다.
