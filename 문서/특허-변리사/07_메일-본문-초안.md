# 변리사 메일 본문 초안 (시설물 기능 특허 검토)

제목 예시: `[SpotVibe] 화장실·운동기구·소화기·공공쓰레기통 찾기 기능 특허 검토 요청 (구현/실시예 분리안 포함)`

화장실·공원 운동기구·소화기·공공 쓰레기통(분리수거함) 찾기 기능은 단순 지도 표시가 아니라, 사용자 제보 검증(메타/AI)·허위신고 임계 정화·긴급상황(화재) 연계를 포함한 공익형 위치정보 처리 파이프라인으로 정리 가능합니다. 다만 현재는 UI/흐름 구현과 백엔드 자동화 실시예가 혼재되어 있어, 출원 시에는 구현 완료 항목과 확장 실시예를 분리해 청구범위를 조정해 주시면 좋겠습니다.
