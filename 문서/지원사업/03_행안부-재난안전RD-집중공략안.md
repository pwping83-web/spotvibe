# SpotVibe — 행안부 재난안전 R&D 집중 공략안

> 작성일: 2026-05-08  
> 대상 사업: 2026년도 재난안전 연구개발사업 신규과제 (행정안전부, 연내 순차 공모 예정)  
> 우선순위: ⭐ 1순위 — 규모(478억) + 적합도 최고 + 연내 기회 남아 있음

---

## 왜 행안부 재난안전 R&D가 1순위인가

1. **예산 규모**: 2026년 신규 과제 총 29건 중 15건 공모 완료, **14건 아직 순차 공모 예정**
2. **예산 확대**: 전년 228억 → 478억 (2배 이상 확대) — 기회 창구 넓어짐
3. **트랙 직접 적합**: "지역맞춤형 재난안전 문제해결 기술개발 지원 (6건)" 트랙이 SpotVibe 성격과 정확히 일치
4. **소프트웨어 과제 선호 증가**: 하드웨어 중심에서 AI·데이터 기반 소프트웨어 과제 비중 확대 추세

> **"하이퍼로컬 일상 플랫폼 기반 시민 초기 대응 SOS 시스템 실증 연구"**  
> 부제: 유령 앱 현상 극복을 위한 DAU 연동 재난 보조망 기술 개발 및 지역 실증

### 핵심 차별화 논리 (심사위원 설득 포인트)

**"기존 SOS 앱과 무엇이 다른가?"**

| 구분 | 기존 SOS 전용 앱 | SpotVibe |
|------|-----------------|----------|
| 일상 사용 동기 | 없음 | 군집 밀도 탐색 (매일 접속) |

| 수익·지속 모델 | 없음 (예산 의존) | B2G SaaS 구독 (지자체 관제) |

## 세부 연구 과제 구성 (예시)

### 과제명
지역 기반 일상 플랫폼 연계 시민 초기 대응 네트워크 기술 개발 및 실증

### 연구 목표
1. 일상 군집 탐색 기능과 SOS 연동 시스템의 **기술 완성도 고도화** (TRL 5 → 7)
2. 실증 지역(1개 구·군 단위) **실사용자 모집 및 KPI 측정**
3. 지자체 재난상황실 연동을 위한 **B2G 관제 대시보드 프로토타입** 개발

### 세부 연구 내용

| 세부 과제 | 내용 |
|-----------|------|
| ① SOS 알림 고도화 | 반경 3km bbox → 지도 기반 폴리곤 필터, 이웃 응답률 피드백 루프 |
| ② AI 허위 신고 고도화 | 비전 LLM 정확도 개선, 집단 신고 자동화 임계 최적화 |
| ③ 해결 사례 사진 연동 | 과거 해결 사례 → 다음 SOS 피어 시트 노출 (사회적 증거 파이프라인) |
| ④ 히어로 인증·게이미피케이션 | 인증서 발급, 등급제, SNS 공유 UX 구현 |
| ⑤ 지자체 관제 API | 집계 데이터, 감사 로그, 알림 연동 API (개인 식별 없음) |
| ⑥ 실증 데이터 분석 | DAU·SOS 발신·응답률·대응 시간·바이럴 확산 지표 수집·분석 |

### 실증 설계

- **실증 지역**: 수도권 1개 구(예: 관악구, 광진구 등 유동인구 많은 지역)
- **실증 기간**: 6개월 (Phase 2)
- **목표 사용자**: DAU 5,000명, SOS 월 50건 이상
- **지자체 협력**: 구청 재난안전과 MOU → 집계 데이터 공유, 현장 피드백
