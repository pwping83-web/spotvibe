## 핵심 기능 (현재 실제 작동 중)

| 기능 | 내용 |
|------|------|
| 📍 실시간 인파 지도 | 연령대별 익명 군집 밀도 시각화, AI 코스 추천 |
| 🆘 SOS 이웃 신호 | 화재·치안·조난·구급 4유형 / 반경 3km 실시간 전파 / 15분 자동 만료 |
| 🧯 소화기 위치 조회 | 시민 GPS+사진 제보 → 지도 표시 → 즉시 조회 |
| 🚻 무료 화장실 | 동일 방식, 반경 500m 즉시 표시 |
| 🦯 취약계층 도움 | 장애인·이동약자 이웃 도움 메시지 MVP 작동 중 |
| ⚠️ 재난문자 연동 | 행안부 긴급재난문자 자동 수집·표시 |
| 🔐 AI 3중 검증 | 텍스트 LLM + EXIF 무결성 + 비전 LLM (허위 신고 자동 차단) |

## 왜 사람들이 앱을 여는가

```
[일상] "오늘 어디 가지?" → AI가 코스 추천
[현장] 야구장·축제 줄 얼마나 긴지 → 현장 제보 실사진
[급박] 화장실 급할 때 → 3초 내 가까운 화장실 표시
[위급] 사고·위협 목격 → SOS 발신 → 반경 3km 이웃에게 전달
```

**20% 마중물 이론**: 100명 중 20명만 앱으로 우회해도 이태원·골목 병목 위험 40% 감소
