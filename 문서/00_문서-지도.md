## 현행 코드 기준 — 명세·기획 통일 주의사항

> 문서 간 불일치가 남아 있을 수 있습니다. 아래 기준을 항상 우선합니다.

| 항목 | 기준값 |
|------|--------|
| SOS 유형 | **4종**: fire / public_safety / missing / medical |
| SOS 반경 | **3km** (`RADIUS_KM = 3` in `useSosSignals.ts`) |
| SOS 만료 | **15분** (`SOS_EXPIRE_MINUTES = 15`) |
| 지도 엔진 | **Leaflet + Carto** (Kakao Maps는 로드맵 TODO) |
| 모더레이션 | 현장제보: allow/held/block / SOS메모: allow/block |
| 인증 | **Supabase Auth + 카카오 OAuth** |
| AI Edge | Groq (LLM·비전), Supabase Edge Functions (Deno) |
