## 1. 서비스 개요

**스팟바이브**는 모바일 웹 기반의 **지역·실시간 탐색 서비스**입니다. 사용자가 현재 위치 또는 관심 지역을 중심으로 **주변 핫스팟(활기 있는 구역)** 과 **이벤트·행사 정보**를 지도와 리스트로 확인하고, 이동 수단에 맞는 안내·맞춤 알림 등 **개인화된 탐색 경험**을 제공합니다.

---

## 2. 제공 기능 (핵심)

| 구분 | 내용 |
|------|------|
| 지도 | OpenStreetMap 기반 지도에서 내 위치, 연령대·군집 시각화, 이벤트 핀 표시 |
| 이벤트 | 지역 행사·버스킹·모임 등 이벤트 목록 및 필터 |
| 알림 | 사용자 설정(선호 시간대 등)에 맞춘 알림·타임라인형 UI |
| 마이페이지 | 연령대·성별·MBTI·활동 관심사·이동 수단(자차/PM/도보 등) 설정 |
| 로그인 | **카카오 계정으로 간편 로그인** 후 설정·프로필 동기화(Supabase 인증 연동) |

---

## 3. 카카오 로그인 도입 목적

- **회원 식별:** 동일 사용자가 여러 기기에서도 설정·프로필을 유지하기 위함  
- **온보딩 단순화:** 별도 이메일·비밀번호 가입 없이 카카오 계정으로 서비스 이용 시작  
- **이용자 보호:** 인증은 Supabase(Auth)를 통해 처리하며, **카카오에서 제공하는 동의 범위 내** 정보만 활용  

**카카오 API로 수집·이용하는 정보(동의 항목에 따름):**  
회원 식별에 필요한 최소 범위(예: 회원번호 등) 및 서비스에 명시된 동의 항목.  
불필요한 민감정보는 수집하지 않으며, 마케팅·제3자 제공은 별도 동의 없이 하지 않습니다.

---

## 3-1. 수집 사유 (카카오 콘솔 · 개인정보 동의항목용)

### 한 줄·짧은 버전 (입력칸이 짧을 때)

이용자 동의 시 성별·연령대는 지도·이벤트 맞춤 탐색 및 마이 설정 반영에만 사용하며, 마케팅·제3자 제공에는 사용하지 않습니다.

### 상세 버전 (수집 사유 텍스트 영역용)

스팟바이브는 지역 기반 탐색 서비스로, **이용자가 카카오 로그인 시 선택 동의한 성별·연령대 정보**를 다음 목적으로만 수집·이용합니다.

1. **서비스 제공:** 지도·이벤트 화면에서 연령대·성향 관련 시각화·필터와 맞춤 안내를 제공하기 위함입니다.  
2. **설정 편의:** 마이페이지에서 사용자가 직접 입력하는 연령대·성별 설정과의 일관성을 돕기 위한 참고(선택 동의 범위 내)입니다.  
3. **제한:** 위 목적 외 광고성 메시지 발송, 제3자 판매, 카카오톡 친구 목록 연동 등 **본 서비스와 무관한 용도에는 사용하지 않습니다.**  
4. **보관:** 관련 법령 및 서비스 개인정보 처리방침에 따른 기간 동안만 보관하며, 목적 달성 시 지체 없이 파기합니다.

## 4. 신청 사유 및 활용 시나리오

### 4-1. 신청 사유

스팟바이브는 **모바일 웹**으로 제공되는 지역 탐색 서비스로, 이용자에게 **별도 회원가입 폼 없이** 안전하게 로그인할 수 있는 수단이 필요합니다. 카카오 로그인을 도입하는 이유는 다음과 같습니다.

1. **이용자 편의:** 이메일·비밀번호 기억 부담을 줄이고, 이미 보유한 카카오 계정으로 즉시 서비스를 시작할 수 있게 합니다.  
2. **계정 일관성:** 기기를 바꿔도 동일한 카카오 계정으로 재로그인하면 **마이페이지 설정(이동 수단, 선호 시간, 관심 태그 등)** 이 서버(Supabase)에 동기화되어 유지됩니다.  
3. **보안·운영:** 인증 처리는 Supabase Auth와 OAuth 표준 흐름을 따르며, 카카오에서 제공하는 **동의한 범위의 정보만** 서비스 운영에 사용합니다.

### 4-2. 활용 시나리오

| 단계 | 이용자 행동 | 서비스 처리 |
|------|-------------|-------------|
| ① | 앱 접속 후 로그인 화면에서 **「카카오로 시작하기」** 선택 | 카카오 OAuth 화면으로 이동, 이용자 동의 후 인가 코드 수신 |
| ② | 카카오 로그인 및 동의 완료 | Supabase Auth 콜백으로 돌아와 세션 발급, **회원 전용 화면**(지도·이벤트·마이) 이용 가능 |
| ③ | 마이페이지에서 이동 수단·선호 시간·관심 태그 등 수정 | 로그인된 사용자 ID 기준으로 DB에 저장, 다음 접속 시에도 동일 설정 유지 |
| ④ | 로그아웃 | 서버 세션 종료, 재접속 시 다시 카카오 로그인으로 복귀 |

**요약:** 카카오 로그인은 **회원 식별·설정 저장·재방문 시 연속된 경험 제공**에만 사용하며, 카카오 친구 목록·메시지 발송 등 **본 서비스와 무관한 용도로는 사용하지 않습니다.**
